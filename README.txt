This school management C++ project for beginners contains both student and teacher record management system.In which a menu is consist of two options to enter data for student or teacher. 
It has two structures one for student and one for teacher's data.s option for student, teacher or to end program.

When you save the data in student then the text file names as student will create which have data of student which you enter recently and same happens when you store the data of teacher.

This project includes following concepts:

1) file handling <fstream>
2) struct
3) Loops
4) Switch
5) if else statement
6) string class <string>
